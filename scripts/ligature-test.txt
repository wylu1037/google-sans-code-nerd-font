Google Sans Code Nerd Font - Ligature Test Pattern
================================================

Test these ligatures in your editor with Google Sans Code NF:

Arrow operators:
-> => <-> <=> <-> ==> <== =<< ==<

Comparison operators: 
== != <= >= === !== <<< >>> <=> =/=

Logic operators:
&& || !! ?. ?: ?? ?: ?:

Math operators:
+= -= *= /= %= **= ^=

Pipes and composition:
|> <| |= ||= ||| ||| |||= 

Comments and documentation:
/* */ /** */ /// // 

Other programming symbols:
.. ... .= :: ::: <: :> << >> <<< >>> 
<$ $> <+ +> <* *> </ /> </> 
<- <-- --> -< -<< >- >>- >>= =<<

HTML/XML:
</ /> <!-- --> <!--- --->

If these ligatures display as single connected glyphs, 
the font ligature support is working correctly!
